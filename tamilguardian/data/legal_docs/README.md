# Legal Documents for RAG

Place your legal documents here for the RAG system:

## Required Documents:
- Tamil Nadu Shops and Establishments Act, 1948
- Tamil Nadu Consumer Protection Rules
- Indian Penal Code (relevant sections)
- Tamil Nadu Civil Courts Act
- Property laws for Tamil Nadu
- Labor laws and employment acts
- Traffic and motor vehicle laws

## Supported Formats:
- PDF files (.pdf)
- Word documents (.docx)
- Text files (.txt)

## File Naming Convention:
- tn_shops_establishments_act.pdf
- consumer_protection_act_2019.pdf
- indian_penal_code.pdf
- tn_civil_courts_act.pdf

The RAG system will automatically process these documents and create embeddings for legal query matching.
