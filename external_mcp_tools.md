# External MCP Tools for NammaLaw AI

## Available MCP Tools:

### 1. **ElevenLabs MCP** (Already integrated)
```bash
pip install elevenlabs-mcp
```

### 2. **Brave Search MCP**
```bash
pip install mcp-server-brave-search
```
- Real-time web search for legal updates
- Case law research

### 3. **GitHub MCP**
```bash
pip install mcp-server-github
```
- Access legal code repositories
- Legal document templates

### 4. **SQLite MCP**
```bash
pip install mcp-server-sqlite
```
- Legal case database
- Precedent storage

### 5. **Filesystem MCP**
```bash
pip install mcp-server-filesystem
```
- Legal document management
- File operations

### 6. **Memory MCP**
```bash
pip install mcp-server-memory
```
- Remember user preferences
- Case history tracking

### 7. **Fetch MCP**
```bash
pip install mcp-server-fetch
```
- Download legal documents
- API integrations

## Integration Steps:

1. Install the MCP server:
```bash
pip install <mcp-package>
```

2. Add to your MCP manager in `base.py`

3. Configure in environment variables

4. Test integration

## Recommended for Legal AI:
- **Brave Search**: Legal research
- **Memory**: User session tracking  
- **Filesystem**: Document management
- **SQLite**: Case database
